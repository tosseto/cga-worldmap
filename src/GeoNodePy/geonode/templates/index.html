WorldMap (alpha) is an Open Source platform for collaborating with maps and spatial data. The platform is being built to assist academic research and teaching, but we hope many others find it useful too.

At it's core, WorldMap lets you create online maps, load data to them, and control who has access to them.

Get started by {% endblocktrans %} {% blocktrans %} browsing existing maps or creating a new map. {% endblocktrans %}
